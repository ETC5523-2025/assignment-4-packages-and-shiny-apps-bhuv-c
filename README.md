# R package

Interactive Shiny dashboard packaged as an R package for ETC5523 (Monash University).  
It explores **Disability-Adjusted Life Years (DALYs) per 100,000** for five healthcare-associated infections across two studies (German PPS, ECDC PPS), split into **YLD** (Years Lived with Disability) and **YLL** (Years of Life Lost).
